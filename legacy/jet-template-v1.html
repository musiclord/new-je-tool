步驟三：預先篩選程序 (Pre-screening)
執行標準化的預先篩選程序，協助查核團隊識別潛在高風險分錄。
ℹ️ 執行預先篩選程序的目的，係協助查核團隊進一步定義出特定的高風險條件，以辨認不尋常的分錄進行測試。這些程序個別的用途僅作為初步判斷使用，其本身將不會是實際設定之高風險範圍條件。
標準化預先篩選程序
#1. 於期末財務報表準備期間核准之分錄
將列出在期末財務報表準備期間開始日及之後核准的分錄
—
#2. 分錄摘要出現特定描述
搜尋包含特定描述文字：Adj, Rev, Reclass, Suspense, Error, Wrong, 調整, 迴轉, 沖銷, 重分類, 避險, 重編, 錯誤, 計畫外, 預算外
—
#3. 未預期出現之特定借貸組合
列出分錄貸方為收入，但借方不是應收帳款、現金或預收款之傳票
—
#4. 分錄金額中有連續 0 的尾數
依 JE 母體中借方平均金額決定 0 的尾數位數（例如平均金額 2,391,281 → 篩選有 000,000 的連續尾數）
—
#5. 依分錄編製者彙總分錄
依分錄建立人員彙總，列出各人員的分錄筆數及金額
—
#6. 較少使用之科目
找出在總帳期間使用次數低於平均值的科目所對應之分錄
—
